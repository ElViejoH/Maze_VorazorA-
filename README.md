# Maze Voraz or A*

Aplicacion educativa para visualizar busqueda de caminos en una grilla y comparar Busqueda Voraz contra A*.

## Version web

Abre `index.html` directamente en el navegador.

## Version desktop

Desde la raiz del proyecto:

```powershell
python .\desktop\main.py
```

## Caracteristicas

- Edicion interactiva de punto A, punto B y obstaculos.
- Comparacion entre Voraz y A*.
- Animacion de celdas visitadas y ruta final.
- Metricas de visitados y pasos de ruta.
- Interfaz refinada con enfoque de dashboard.

Para mas contexto del desarrollo, revisa `CONTEXTO_PROYECTO.md`.
